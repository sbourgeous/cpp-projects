# Python Fundamentals: Hello World

This project provides an introductory exercise in creating and running a basic C++ program.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Create a folder or a repository
2. Create a C++ file (e.g. main.cpp) inside your folder/repository
3. Write and run code

Instructions for using the software:

1. Have C++ add on installed on your system.
2. Run the main.cpp file
3. The program will display the message **Hello World!** in the terminal.
