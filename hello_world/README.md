# Python Fundamentals: Hello World

This project provides an introductory exercise in creating and running a basic Python program. It optionally includes website links if you want to create and add your code to a GitHub repository or add a markdown file.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Create a folder or a repository
2. Create a Python file (e.g. main.py) inside your folder/repository
3. Write and run code

Instructions for using the software:

1. Have Python installed on your system.
2. Run the main.py file
3. The program will display the message **Hello World!** in the terminal.
